# Build
## Example
```
mkdir build
cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
make
sudo make install
```

## cpp-netlib
```
git clone git://github.com/cpp-netlib/cpp-netlib.git
cd cpp-netlib
git submodule init
git submodule update
mkdir build
cd build
cmake -DCMAKE_BUILD_TYPE=Release -DCPP-NETLIB_BUILD_DOCS=OFF ..

```
